## `7ccd1640` 提取内容

### 版本号脚本

- 新增 `scripts/bump-version.js`。
- 新增 `package.json` 脚本：
  - `bump-version`: `node ./scripts/bump-version.js`
- 新增必要 devDependency：
  - `enquirer`
  - `picocolors`

实现说明：

- 读取 `manifest.config.ts` 中的 `versionName` / `versionCode`。
- 支持 `--dry-run`，模拟运行不写文件。
- 支持 `--type=patch|minor|major|none`。
- 非交互环境且未指定 `--type` 时，默认只升级 `versionCode`，不修改 `versionName`。
- 交互环境下可选择 patch/minor/major/none/cancel。

### Eruda H5 调试插件

- 新增 `scripts/vite-plugin-eruda.js`。
- `vite.config.ts` 引入 `vitePluginEruda`。
- 只在 `UNI_PLATFORM === 'h5' && mode === 'development'` 时注入 Eruda。
- H5 production 和非 H5 平台不注入 Eruda。
- Eruda 默认从 `https://cdn.jsdelivr.net/npm/eruda` 加载；内网环境如果访问失败，可通过 `erudaUrl` 改为内网镜像。
